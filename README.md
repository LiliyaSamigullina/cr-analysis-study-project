# Conversion Dashboard
## Project Description
Dashboard for analyzing conversion rates from visits to registrations. Data is retrieved via an API, processed, and visualized using Python.
## Technology Stack
- Python
- Pandas
- Matplotlib
- Seaborn
- Requests
- dotenv (for environment variables management)
